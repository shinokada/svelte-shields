# Svelte Shields

Shields badge components for Svelte.

## Docs

[Svelte Shields Docs](https://svelte-shields.codewithshin.com)

## Requirements

- Svelte 5 (Runes)

## Installation

```sh
pnpm i -D svelte-shields
```

## Components

- GitHub for GitHub release
- GitHubDownloads
- JsrVersion
- NpmAuthorDownloads
- NpmDownloads
- NpmVersion
- PypiVersion
- StaticBadge
